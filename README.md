This repository contains the Apps Script projects for hospitality booking platforms, dashboards and shared operational tools.

The root batch files are kept as launch shortcuts. They point into the new folders, so you can keep using `mnkpush.bat`, `demopush.bat`, `clientpush.bat`, `hospitalitypush.bat` and the shared tool push scripts from this top-level folder.
