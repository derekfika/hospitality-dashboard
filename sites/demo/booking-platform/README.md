# Demo Hospitality Booking Platform

Unbranded client-facing hospitality booking platform for demos.

This project keeps the current booking flow, menu schema, direct-dashboard write path, multi-select menu options, and polished UI behaviour. The identity, colours, spreadsheet IDs and email defaults have been neutralised for demo use.

## Demo Defaults

- Client-facing name: `Demo Hospitality`
- Booking reference prefix: `DEMO`
- Site logo fallback: `YOUR LOGO HERE`
- Palette: neutral greys and soft warm paper
- Booking spreadsheet ID: blank
- Dashboard spreadsheet ID: blank

Before deploying, create/link a new Apps Script project and add fresh spreadsheet IDs in `00_Config.js` or the Settings sheet.
