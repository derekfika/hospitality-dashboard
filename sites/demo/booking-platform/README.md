# FIKA Hospitality Booking Platform

Client-facing FIKA hospitality booking platform for sales demos and internal walkthroughs.

This project keeps the current booking flow, menu schema, direct-dashboard write path, multi-select menu options, feedback preview, and polished UI behaviour. The visible identity, colours, and defaults are FIKA branded.

## FIKA Defaults

- Client-facing name: `FIKA Hospitality`
- Booking reference prefix: `DEMO`
- Primary logo: FIKA logo URL
- Site fallback: `Hospitality`
- Palette: FIKA Purple with turquoise, green, yellow, and orange accents
- Feedback recipient: `derek@fikacatering.com`

Before deploying, create/link an Apps Script project and add the spreadsheet IDs in `00_Config.js` or the Settings sheet.
